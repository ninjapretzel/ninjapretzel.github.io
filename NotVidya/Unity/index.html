'Tiles'
v1 v2 v3
2D Tile-based game in Unity.
A simple game I made that uses an Orthographic mode camera to render a scene 'pixel-perfect' to the screen. Uses a sprite animator and an adapted version of the 2dplatform movement. I do a little bit of procedural generation to randomize the look of the world. I am currently working on optimizing the rendering of worlds, right now each 'tile' is its own object with its own collider and renderer. This is incredibly inefficient. I want to make each room (loaded from a file) create its own colliders, and one renderer, in the most efficient way possible on load.
